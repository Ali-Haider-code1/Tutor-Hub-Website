# Tutor-Hub-Website
This website is develope using HTML,CSS, Bootstrap on backend i use NodeJs and for DataBase i used MongoDB

# 1: Install Node and MongoCompass on your System
# 2: Install all resources using      npm install
# 3: Add your database in mongoose DB link 
# 3: Run command    node index.js on terminal
# 4: Open localhost:3000 and run the website.
